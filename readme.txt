Website Inves Tani Laravel
Website Investani merupakan website yang berfungsi sebagai tempat orang-orang menginvestasikan dana mereka pada sebua project pertanian/peternakan.

Fitur 
Websete ini memiliki 1 administrator, dan 1 (user)pengguna yang memiliki role ganda yaitu sebagai Pendonasi dan Penerima Donasi
yang tersedia fitur sebagai berikut :
1. Administrator
	> Mengelola keseluruhan data yang ada
	> Mengelola data user
	> Mengelola data investasi
2. User
	> Posting data investasi
	> Melihat data investasi
	> Kelola postingan data investasi
	

Teknologi Yang Digunakan
	Backend : PHP v 7.3
	Framework PHP : Laravel 7
	Dashboard Template : Platform Crowdfunding
